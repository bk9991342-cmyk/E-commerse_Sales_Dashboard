# E-commerse_Sales_Dashboard
## Project Objective
The objective of this project is to analyze e-commerce sales data stored in Excel and create an interactive Power BI dashboard that provides insights into sales performance, customer trends, and product-wise analysis. The dashboard enables stakeholders to track key KPIs and make data-driven business decisions.

## Dataset Used
- <a href="https://github.com/bk9991342-cmyk/E-commerse_Sales_Dashboard/blob/main/Details.csv">Dataset-1</a>
- <a href="https://github.com/bk9991342-cmyk/E-commerse_Sales_Dashboard/blob/main/Orders.csv">Dataset-2</a>

## Process
- Data cleaning in Excel → Data loading in Power BI → KPI creation → Dashboard visualization → Insights generation.
